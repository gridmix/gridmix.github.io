Gridmix is an independent continuation of the [Gridsome](https://github.com/gridsome/gridsome) Vue.js static site generator, modernized for current Node.js and tooling. Not affiliated with (_or endorsed by_) the original Gridsome maintainers.

Check out [the website repository](https://github.com/gridmix/gridmix-website) and [the main project repository](https://github.com/gridmix/gridmix) for more detail.
